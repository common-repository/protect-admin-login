== Description ==

 A simple plugin allows to overwrite wp-admin url to login backend.
 
 If you run a WordPress website, you must use “Protect Admin Login” to secure it against hackers.
 
 It blocks default wp-admin login link.
 
 You can add new url in backend in Settings/Permalinks section.

== Installation ==
1. Upload plugin directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go thorugh Settings/Permalinks and add new slug to replace wp-admin.


== Frequently Asked Questions ==

=  How can I use this plugin? = 

1. The plugin Settings/Permalinks will be in the setting section of backend. 

2. Set new slug to login. Logout from everywhere and login with new slug.
